## Overview

The website counts with a landing page where the main info is and two sections. One for invitees to confirma assistance and another for the couple to check the confirmations and info such as selected menus.

### The challenge

-Build a page from scratch
-Create a selection menu for invitees to finde themselves by name and access a form where they should complete their personal info and menu selection. This will be shown to them and added to the couple´s list of information which they can access through their own section.
